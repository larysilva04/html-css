o que é css?

css ou cascading style sheets (folhas de estilo em cascata, em portugues), é uma linguagem de estilo usada para descrever a apresentação de documentos escritos em HTML ou XML. O css permite que os desenvolvedores separem a estruturas e o conteúdo do deing e da formatação de um site. com o css, é possível controlar aspctos como cores, fontes,espaçamentos e loyouts de uma pagina web. ele ofereceum modo eficiente e flexivel para aplicar estilos a varios elementos simultaneamente e ajustar a aparência de um paginas sem alterar o seu conteudo HTML. O css pode ser usado para estilizar um documento muito básico de texto - por exemplo, alterando a cor e tamanho dos titulos e links. Pode ser usado para criarr loyout - por exemplo, transformando uma simples coluna de um texto em um layout com uma aárea de conteudo prinipal e um sidebar (barra lateral) para as informaçoes relacinadas. pode ate ser usado para efeitos tais como animação. dê um olhada nos links deste paragrafos, para ver exemplos especificos.
